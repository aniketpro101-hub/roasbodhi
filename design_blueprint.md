## ⚡ ANIMATIONS & INTERACTIONS

```
ANIMATION SYSTEM:

GUIDING PRINCIPLE:
├── Purposeful (every animation adds meaning)
├── Fast (150-300ms max for UI interactions)
├── Not distracting (content is king)
├── Respects prefers-reduced-motion
└── CSS-only where possible (no heavy JS libraries)

GLOBAL ANIMATIONS:

1. PAGE LOAD:
├── Fade in + slide up (50px → 0)
├── Duration: 600ms
├── Easing: cubic-bezier(0.16, 1, 0.3, 1) (smooth)
└── Stagger: 100ms between elements

2. SCROLL REVEAL:
├── Intersection Observer API (lightweight)
├── Elements: Fade + slide up on enter viewport
├── Threshold: 15% visible
├── One-time: Animation doesn't repeat on scroll back
└── CSS classes: .reveal → .revealed

3. HERO FLOATING ELEMENTS:
├── Dashboard mockup: Float animation
│   └── @keyframes: translateY(0) → translateY(-12px) → translateY(0)
│   └── Duration: 6s, infinite, ease-in-out
├── Glow blobs: Slow rotation
│   └── Duration: 20s, infinite, linear
└── Particles: Random float paths, 15-30s duration

4. COUNTER ANIMATION:
├── Triggers: When counter enters viewport
├── Duration: 2000ms
├── Easing: Ease-out
├── Method: requestAnimationFrame (smooth, no jank)
└── Format: Adds ₹, +, X suffix automatically

5. BUTTON INTERACTIONS:
├── Hover: translateY(-2px) + shadow increase
├── Active: translateY(0) + shadow decrease
├── Primary CTA pulse: 
│   └── Glow expands and fades every 4s
│   └── @keyframes: box-shadow 0 → 0 0 0 15px rgba(saffron,0)
└── Transition: 200ms ease

6. CARD HOVER:
├── Transform: translateY(-8px)
├── Shadow: Increases depth
├── Border: Color appears
└── Transition: 300ms cubic-bezier(0.4, 0, 0.2, 1)

7. NAVIGATION DROPDOWN:
├── Trigger: Hover
├── Show: Opacity 0→1 + translateY(-8px→0)
├── Duration: 200ms
└── Hide: Reverse, 150ms

8. TESTIMONIAL SLIDER:
├── Auto-play: 5s interval
├── Transition: Horizontal slide (CSS transform)
├── Swipe: Touch support (native CSS scroll-snap)
└── No external Swiper.js needed

9. MORPHING TEXT (Hero):
├── Word cycling animation
├── Current word: fade out + slide up
├── New word: fade in + slide up from bottom
├── Duration: 400ms per transition
├── Pause: 2000ms between changes
└── Pure CSS with animation-delay staggering

PERFORMANCE RULES:
├── ONLY use: opacity, transform, filter (GPU-accelerated)
├── NEVER animate: width, height, top, left, margin (causes reflow)
├── Use will-change: transform sparingly
├── Debounce scroll listeners
└── Remove will-change after animation completes
```
